# React + Vite
Basic React + Vite setup with auto-reload and simple code checks.

Plugins you can use:
@vitejs/plugin-react → Babel for fast refresh

@vitejs/plugin-react-swc → SWC for fast refresh

How to run
Open the project folder in VS Code
Make sure Node.js is installed

In the terminal, run:
npm install
npm run dev


Open the link in your browser (usually http://localhost:5173)
